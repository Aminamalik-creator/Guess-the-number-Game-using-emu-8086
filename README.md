# 🎯 Guess the Number Game (EMU8086)

> A simple yet engaging number guessing game built using 8086 Assembly Language in EMU8086.

---

## 📌 Overview
This project is a console-based game where the computer generates a number and the user tries to guess it. After each attempt, the program provides hints such as **"Too High"** or **"Too Low"**, guiding the player toward the correct answer.

---

## ⚙️ Features
✨ Interactive user input  
✨ Real-time feedback on guesses  
✨ Efficient number comparison logic  
✨ Lightweight and fast execution  

---

## 🛠️ Technologies Used
- 8086 Assembly Language  
- EMU8086 Emulator  

---

## ▶️ How to Run
1. Open the project in EMU8086  
2. Compile the code  
3. Run the program  
4. Enter your guesses and follow the hints  

---

## 🎮 Game Logic
- The system selects a number  
- User inputs a guess  
- Program compares input with the target number  
- Feedback is displayed until the correct number is guessed  

---

## 📊 Learning Outcomes
- Understanding low-level programming  
- Working with registers and memory  
- Implementing logic in Assembly Language
